# Калькулятор здоровья

Интерактивное веб-приложение для расчета индекса массы тела (BMI) и дневной нормы калорий.

## Функциональность

### 1. Расчет индекса массы тела (BMI)
- Ввод роста и веса
- Автоматический расчет BMI
- Категоризация результата:
  - **Недостаточный вес** (BMI < 18.5)
  - **Нормальный вес** (BMI 18.5 - 24.9)
  - **Избыточный вес** (BMI 25 - 29.9)
  - **Ожирение** (BMI ≥ 30)
- Рекомендации по каждой категории

### 2. Расчет дневной нормы калорий
- Ввод параметров: возраст, пол, рост, вес
- Выбор уровня активности
- Расчет базового метаболизма (BMR) по формуле Harris-Benedict
- Расчет дневной нормы калорий
- Распределение макронутриентов:
  - Белки (25%)
  - Жиры (30%)
  - Углеводы (45%)

## Технологический стек

- **HTML5** - структура
- **CSS3** - стили с адаптивным дизайном
- **JavaScript (Vanilla)** - логика расчетов

## Классы JavaScript

### BMICalculator
- `calculate(heightCm, weightKg)` - расчет BMI
- `getCategory(bmi)` - определение категории
- `getCategoryText(category)` - текстовое описание категории
- `getDescription(category)` - полное описание рекомендаций

### CalorieCalculator
- `calculateBMR(age, gender, heightCm, weightKg)` - расчет базового метаболизма
- `calculateDailyCalories(bmr, activityLevel)` - расчет дневной нормы калорий
- `getMacroNutrients(calories)` - распределение макронутриентов

### UIController
- Управление интерфейсом
- Переключение между вкладками
- Валидация и обработка форм
- Отображение результатов

## Использование

1. Откройте `index.html` в браузере
2. Перейдите на необходимую вкладку
3. Введите требуемые параметры
4. Просмотрите результаты расчетов и рекомендации

## Адаптивный дизайн

Приложение полностью адаптировано для:
- Десктопных браузеров
- Планшетов
- Мобильных телефонов

## Важно

Это приложение предназначено в образовательных целях. Результаты не являются медицинским советом. Для получения персональных рекомендаций проконсультируйтесь с врачом.
