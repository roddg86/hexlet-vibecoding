# 🏥 Калькулятор здоровья

Полнофункциональное веб-приложение для расчета основных показателей здоровья и питания. Включает четыре независимых калькулятора с рекомендациями на основе индивидуальных данных.

**🌐 Живой демо:** [https://roddg86.github.io/hexlet-vibecoding](https://roddg86.github.io/hexlet-vibecoding)

## ✨ Функциональность

### 📊 1. Индекс массы тела (BMI)

Быстрый расчет индекса массы тела и оценка веса.

**Входные данные:**
- Рост (см): 100-250
- Вес (кг): 30-300

**Результаты:**
- Значение BMI (кг/м²)
- Категория веса с цветовым кодированием:
  - 🔵 **Недостаточный вес** (BMI < 18.5)
  - 🟢 **Нормальный вес** (BMI 18.5 - 24.9)
  - 🟠 **Избыточный вес** (BMI 25 - 29.9)
  - 🔴 **Ожирение** (BMI ≥ 30)
- Персональные рекомендации для каждой категории
- Real-time расчет при вводе данных

### 🍽️ 2. Дневная норма калорий

Расчет калорийности рациона на основе личных параметров и активности.

**Входные данные:**
- Возраст: 10-100 лет
- Пол: мужской/женский
- Рост: 100-250 см
- Вес: 30-300 кг
- Уровень активности (5 вариантов от сидячего до очень высокого)

**Результаты:**
- Базовый метаболизм (BMR) в ккал/день
- Дневная норма калорий с учетом активности
- Рекомендуемое распределение макронутриентов:
  - 🥚 **Белки** (25% калорий)
  - 🧈 **Жиры** (30% калорий)
  - 🍞 **Углеводы** (45% калорий)
- Граммовое выражение каждого макронутриента

**Формула:** Harris-Benedict для базового метаболизма с коэффициентом активности

### 💧 3. Дневная норма воды

Расчет необходимого количества воды в день.

**Входные данные:**
- Вес (кг): 30-300
- Уровень активности (3 варианта: низкая/умеренная/высокая)

**Результаты:**
- Дневная норма воды в литрах
- Эквивалент в стаканах (250 мл)
- Рекомендуемое расписание приема воды (8 приемов в день)

**Формула:** вес(кг) × 0,035 × коэффициент активности

### 🥗 4. Витамины и минералы

Рекомендуемая дневная норма 8 ключевых витаминов и минералов.

**Входные данные:**
- Вес (кг): 30-300
- Пол: мужской/женский
- Возраст: 1-120 лет

**Рассчитываемые вещества:**
| Вещество | Единица | Функция |
|----------|---------|---------|
| Витамин A | мкг | Зрение, иммунитет |
| Витамин D | мкг | Кости, усвоение кальция |
| Витамин C | мг | Иммунитет, кожа |
| Витамин B12 | мкг | Энергия, нервная система |
| Кальций | мг | Кости, зубы |
| Железо | мг | Кровь, транспорт кислорода |
| Магний | мг | Мышцы, релаксация |
| Цинк | мг | Иммунитет, заживление ран |

**Результаты:**
- Рекомендуемые нормы по стандартам WHO/RDA
- Персональные советы по продуктам питания
- Учет гендерных и возрастных различий

## 📋 Технические детали

### Формулы и источники

**BMI:** вес(кг) / рост²(м)

**Базовый метаболизм (Harris-Benedict):**
- Мужчины: 88.362 + (13.397 × вес) + (4.799 × рост) - (5.677 × возраст)
- Женщины: 447.593 + (9.247 × вес) + (3.098 × рост) - (4.330 × возраст)

**Дневные калории:** BMR × коэффициент активности

**Вода:** вес(кг) × 0.035 × коэффициент активности

**Витамины/минералы:** Рекомендации WHO/RDA по возрасту и полу

### Граничные значения и валидация

| Параметр | Минимум | Максимум | Единица |
|----------|---------|----------|---------|
| Рост | 100 | 250 | см |
| Вес | 30 | 300 | кг |
| Возраст | 1 | 120 | лет |
